so there may be a couple of glyphs that are not quite correct.



ABOUT THE FONT

In the steel alphabet all double letters are removed.
Certain letters are represented by the same symbol (you can use the normal key, they just map to the same symbol):
o/u pewter
e/i tin
any "k" sound bendalloy (q included)
x is represented by "ks"
Ch is a capital C
Sh is a capital S

The numbers 0-9 print the characters as in the chapter headers in the books



I reccomend you turn off automatic capitalization as it will show English capital letters.

To do so in open office:

Format->AutoCorrect->AutoCorrect options.
In the replace tab, find the rule that converts 'i' to 'I' and delete it. (You can always make it again)
In the options tab, untick "Capitalize first letter of every sentence" in both [M] and [T]





INSTALLING:

Windows vista, 7, 8 & 10:
Method A:
Double click on the font file, then click install (doesn't work on vista). (Click the install button on the popup in Win 10.)

Method B:
Right click the font file and select install

Method C:
Copy the font file into the Fonts folder, it will install automatically

Windows XP:
Start->Control Panel->Appearances and Themes->Fonts
File->Install New Font...
Select the font.
